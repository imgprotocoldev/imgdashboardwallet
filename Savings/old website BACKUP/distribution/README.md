# IMG Protocol - Distribution Analytics Dashboard

## Overview
The Distribution Analytics Dashboard is a premium feature that provides comprehensive analysis of IMG token distribution patterns, wallet statistics, and geographic insights.

## Features

### 📊 Overview Cards
- **Total Distributed**: Shows total IMG tokens distributed across all wallets
- **Active Wallets**: Displays number of actively participating wallets
- **Average Distribution**: Shows average IMG amount per distribution
- **Distribution Rate**: Real-time distribution rate in IMG per hour

### 📈 Charts
- **Distribution Timeline**: Line chart showing distribution activity over time
- **Wallet Distribution**: Doughnut chart showing holder categories
- **Period Selector**: Choose between 24h, 7d, 30d, or 90d views

### 📋 Distribution Metrics
- **Geographic Distribution**: Breakdown by region (North America, Europe, Asia Pacific, Other)
- **Token Holder Categories**: Analysis by wallet size (Whales, Large, Medium, Small)

### 📊 Activity Table
- **Recent Activity**: Table showing recent distribution transactions
- **Transaction Details**: Date, wallet address, amount, type, and status
- **Refresh Functionality**: Manual data updates

### 💡 Distribution Insights
- **Growing Adoption**: Community growth metrics and trends
- **Balanced Distribution**: Token distribution balance analysis
- **Global Reach**: Geographic expansion and user diversity

## Access Control
- **Premium Feature**: Requires wallet connection with 47.5k+ IMG tokens
- **Sidebar Integration**: Accessible via sidebar navigation
- **Wallet Verification**: Automatically checks token balance
